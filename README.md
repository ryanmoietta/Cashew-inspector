# 🥜 Cashew Link Inspection

Cashew Link Inspection is a defensive cybersecurity tool designed to analyze URLs and detect suspicious behavior such as link spoofing, hidden destinations, redirects, and phishing indicators.

## Features

- 🔍 Detects `@` URL spoofing techniques
- 🔗 Expands shortened URLs and reveals redirect chains
- 🚨 Detects suspicious domains and impersonation attempts
- 🧬 Detects lookalike domains and typo tricks
- 🌐 Detects punycode/homograph attacks
- 🛡️ Analyzes suspicious paths and URL patterns
- 📊 Generates a risk score with detailed findings
